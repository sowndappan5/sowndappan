Real-time Object Distance Estimation & Fire Detection

Problem

Early detection of forest fires is critical, but localizing the exact distance of a fire using drones typically requires expensive LiDAR or radar hardware. Monocular cameras are cheap and ubiquitous but lack inherent depth perception, making it difficult to coordinate rapid response teams based on visual feeds alone.

Architecture Diagram

Workflow

The system captures a live camera feed and feeds the frames into a YOLOv8 object detection model specifically trained on forest fire datasets. Once a bounding box is generated, the system uses a novel calibration-based perspective mapping algorithm to translate the 2D pixel coordinates of the bounding box's base into real-world distance metrics. This allows for real-time overlay of distance metrics directly onto the video feed.

Tech Stack

YOLOv8 OpenCV PyTorch Python

Challenges

The primary challenge was devising a mathematical method to accurately extract depth from a single 2D image source without stereo vision. The mapping algorithm had to account for camera angle, lens distortion, and varied terrain while remaining computationally inexpensive enough to run in real-time alongside a deep learning object detector.

Trade-offs

By opting for a mathematical perspective mapping approach over a secondary neural network for depth estimation, the system trades the ability to handle highly irregular, uncalibrated terrains for blazing fast O(1) time complexity. This allows the system to run on standard CPU hardware without a dedicated GPU for depth processing.
